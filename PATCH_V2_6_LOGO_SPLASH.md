# Patch V2.6 — Logo, splash screen et finition visuelle

Ce patch finalise l'identité visuelle de S&S Budget autour du logo validé.

## Contenu

- `assets/icon.png` : icône principale de l'application.
- `assets/adaptive-icon.png` : icône Android adaptative.
- `assets/splash.png` : écran de lancement sur fond bleu nuit.
- `assets/favicon.png` : favicon web/PWA.
- `assets/logo.png` : version réutilisable du logo dans l'interface.
- `app.json` : configuration Expo alignée avec les nouveaux assets.

## Principes retenus

- Aucun texte ajouté dans le logo.
- Fond bleu nuit cohérent avec l'interface actuelle.
- Photo conservée dans un cercle propre.
- Compatible Expo Go, iOS, Android et web/PWA.

## Installation

Copier les fichiers `app.json` et `assets/*` dans le projet, puis relancer Expo avec cache vidé.
